# BMS College of Engineering - Modern Website Redesign

A prestigious, modern redesign of the official B.M.S. College of Engineering (BMSCE) website. This project aims to transform the institution's digital presence into a "Princeton-style" experience, blending academic heritage with cutting-edge web design and interactions.

---

## 🌟 Vision

Our goal is to create a digital home for BMSCE that reflects its status as one of India's premier engineering institutions. The design prioritizes **clarity, prestige, and institutional authority**, featuring:
- **"Princeton-style" Aesthetic**: A clean, balanced layout with a sophisticated color palette (Navy & Gold).
- **Advanced Animations**: Fluid, purposeful micro-interactions and scroll-triggered motion using Framer Motion.
- **Academic Focus**: Dedicated, rich sections for Departments, Research, Innovation, and Placements.

---

## 🚀 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Intelligence**: [Google Gemini AI API](https://ai.google.dev/) (Powers dynamic content and features)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

---

## ✨ Key Features

- **Dynamic Hero Section**: High-impact visual storytelling with smooth transitions.
- **Comprehensive News & Stats**: Stay updated with the latest from the college.
- **Departmental Grid**: Elegant navigation through all engineering branches.
- **Admissions & Placements**: Seamless, user-friendly paths for prospective students and career opportunities.
- **Visionaries & Testimonials**: Showcasing the leadership and the success of our community.
- **Glassmorphic Components**: Modern UI elements with subtle transparency and depth.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- npm or yarn

### Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd browser-battle
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env.local` file in the root directory and add your Google Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **View the app**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.
